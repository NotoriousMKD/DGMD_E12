I made this tool to help plan a garden in the style of square foot gardening. I wrote in my proposal about how I couldn’t find a tool like this when I was starting my garden for the first time last year, but while I was googling to find a list of recommended spacing of various plants for square foot gardening, I came across a similar tool. However, one thing that I dislike about that version is that if forces you to only put one type of plant in each square. When I was initially planning my tool, I thought that was also how I would design mine, but once I started working, I realized that I prefer more flexibility because in real life I sometimes split squares up between multiple types of plants, tuck basil plants in between tomato plants, etc. So, although the tool I built requires more clicking and dragging to populate the whole garden, it also provides more flexibility.

My approach to working on this project was to break it down into the essential bits first and make sure I could get those working as sort of an MVP or proof of concept, then to move on to filling in the gaps. I first tested dragging multiples “plants” (boxes the sizes of the spacing dimensions) on the page, and then looping through an array of fake data that was supposed to replicate user-inputted data to see if I could create all of the shapes I wanted from an array. I think the next thing I did was add the “plants” to the page via button click. After that, I was mostly over the biggest hump. There were a ton of little mistakes and challenges along the way, but one that sticks out is getting the vegetables to resize when the size of the garden is changed. Working with javascript classes is tricky. Also, I initially thought I would draw all of the veggies using p5, but that was an insane idea, so I used icons instead.

For future work, I would provide the option to bulk add, say, an entire square of 16 carrots as opposed to only giving the option of adding one carrot at a time (and so on for other wee vegetables). That would be easy to implement. More difficult work I would like to do is to store the position of the vegetables in local storage and keep them in the right places of the garden on re-size. I briefly attempted this and didn’t really make and forward progress while also managing to break almost everything, so I decided that didn’t need to be a part of this first pass. Another feature I would like to add is a trash can for unwanted veggies!
